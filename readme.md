# Track Share
A web app that makes sharing music between streaming providers easier.

## Setup
```
git clone git@github.com:awesomecodegreatjob/track-share.git
cd track-share
composer install
cp .env.example .env
php artisan key:generate
php artisan serve
```

## Requirements
* PHP 5.6+
* [Composer](https://getcomposer.org/)
